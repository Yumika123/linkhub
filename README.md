## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Functionality

Authorization user can:

- create several pages
- edit pages
- delete pages
- add links to pages
- edit links
- delete links
- add styles to pages

Anonymous user can:

- create anonymous page
- get public access to the anonymous page
- attach to the anonymous page to account
